Token de API

Para proporcionar acesso aos nossos aplicativos móveis e outros aplicativos de terceiros, você precisará primeiro gerar um token de API.

  • Basta clicar em "Criar" para gerar o seu token; depois copie-o e cole-o no aplicativo.
  • Escolha o tipo de token de API específico que você necessita, com base nas capacidades que você desejar disponibilizar.
