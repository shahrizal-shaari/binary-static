Jeton d'API

In order to access our mobile apps and other third-party applications, you'll first need to generate an API Token.

  • Simply click on "Create" to generate your token; then copy and paste it into the app.
  • Choisissez le type de jeton d'API dont vous avez besoin, selon les fonctionnalités que vous souhaitez proposer.
