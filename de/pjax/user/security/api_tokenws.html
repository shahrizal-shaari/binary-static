API Token

Um Zugang zu unseren mobilen Apps und anderen Drittanbieteranwendungen zu haben, müssen Sie erst einen API Token erzeugen.

  • Klicken Sie einfach auf "Erstellen", um Ihren Token zu generieren, dann kopieren Sie ihn und fügen ihn in die App ein.
  • Wählen Sie auf der Grundlage der Möglichkeiten, die Sie gerne zur Verfügung stellen möchten, die spezifische Art des von Ihnen benötigten API-Tokens aus.
