Ключ API

Чтобы получить доступ к нашим мобильным приложениям и другим сторонним приложениям, нужно сначала сгенерировать ключ API.

  • Просто нажмите кнопку "Создать," чтобы сгенерировать ключ; затем скопируйте и вставьте его в приложение.
  • Выберите определённый тип нужного Вам ключа API, согласно параметрам, которые Вы хотите активировать.
